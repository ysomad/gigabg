# GIGA Battlegrounds

Heartstone Battlegrounds like game written in Go using Ebiten engine.

# Documents
- Code style guide `STYLE.md`
- Project architecture `ARCHITECTURE.md`

## Ebiten
- Fonts `https://ebitengine.org/en/examples/font.html`
- Sprites `https://ebitengine.org/en/examples/sprites.html`
- Animations `https://ebitengine.org/en/examples/animation.html`
- Drag and drop `https://ebitengine.org/en/examples/drag.html`
- Performance tips `https://ebitengine.org/en/documents/performancetips.html`
- Use `vector.FillRect` instead of `vector.DrawFilledRect`
- Use `vector.FillCircle` instead of `vector.DrawFilledCircle`
