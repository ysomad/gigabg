# GIGA Battlegrounds

Heartstone Battlegrounds like game written in Go using Ebiten engine.

# Documents
- Code style guide `STYLE.md`
- Performance tips for ebitenengine `https://ebitengine.org/en/documents/performancetips.html`
- Project architecture `ARCHITECTURE.md`
- Drag and drop example `https://ebitengine.org/en/examples/drag.html`

## Ebiten
- Use `vector.FillRect` instead of `vector.DrawFilledRect`
- Use `vector.FillCircle` instead of `vector.DrawFilledCircle`
